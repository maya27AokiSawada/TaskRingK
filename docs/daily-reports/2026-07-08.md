## 技術判断

### Firestore サブコレクションパス

SharedList は `/SharedGroups/{groupId}/sharedLists/{listId}` のサブコレクション構造。
`FirestoreSharedListDataSource` では `collection("SharedGroups").document(groupId).collection("sharedLists")` でアクセスする。

### アイテム差分更新（§12-1）

`addOrUpdateItem` は `update(mapOf("items.${item.itemId}" to item.toItemMap(), "updatedAt" to serverTimestamp()))` のみを送信。
アイテムマップ全体を `set()` することは禁止（並行編集でデータ消失のリスク）。

### Room オプティミスティック更新

`addOrUpdateItem` / `removeItem` では Firestore の Listener が次の emit を返す前に UI が更新されるよう、
`observeById(listId).firstOrNull()` でエンティティを取得し items を更新して即 upsert する。
Firestore の実際のデータが到達した時点で自動的に上書きされる。
